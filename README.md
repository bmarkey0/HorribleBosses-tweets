# HorribleBosses-tweets

Data and code for analyzes #HorribleBosses tweet corpus. 

This project uses the docuscospacy python package to analyze tweets (https://github.com/browndw/docuscospacy). Please refer to docuscospacy API and documentation for troubleshooting. 

As of 01/2026, due to recent changes with pip requirements, the provided call to the docuscospacy model on huggingface does not work. Code must be run in GoogleColabs for model to load correctly.
